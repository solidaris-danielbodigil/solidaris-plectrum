# Animation Guidelines

Animations bring our interfaces to life and help users understand state changes. Properly used animations improve the user experience by providing visual feedback, guiding attention, and creating smooth transitions.

## Overview and Demo

Our animation system is built on CSS custom properties and organized into categories:

- **Duration**: How long an animation takes to complete
- **Timing**: The easing curve that controls the animation's progression
- **Delay**: How long to wait before starting the animation
- **Iteration**: How many times an animation should repeat
- **Direction**: The direction in which an animation plays
- **Type**: Specific transformation or effect types
