Auto Cols

Auto-fit will decide how many cols fit in the wrapper.
With the minmax(0, 1fr) function, all columns will be equal width.

o-grid o-grid--auto-cols

Auto Cols with Min width defined

Auto-fit will decide how many cols fit in the wrapper.
Also renders equal width cols but this time minmax(<\col>, 1fr) accepts a min width.

o-grid o-grid--auto-cols-md

Cols width defined span values & grid responsiveness

This grid is mobile-first, meaning that the code is by default set for small screens : .col-XX will cover all screens.
As soon as a breakpoint is specified as suffix: @xs @sm @md @lg @xl, they will take the lead when their breakpoint is met.
You can define different sizes, order, alignment and distribution on those breakpoints.

.col-12 .col-3@md .col-6@lg
.col-12 .col-3@md .col-6@lg
.col-12 .col-3@md
.col-12 .col-3@md

Place-items

The CSS place-items shorthand property allows you to align items along both the block and inline directions at once (the align-items and justify-items properties).
If the second value is not set, the first value is also used for it.
