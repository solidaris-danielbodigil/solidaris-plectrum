#### Does your shadow means your item is elevated?

- If the answer is YES, use the elevation placeholder tool

- If the answer is NO, use a shadow placeholder

#### Need a more specific shadow?

You can use

```css
@include shadow($_inset: false, $_offset: 2, $_direction: all, $_blur: 2, $_spread: none, $_color: default, $_secondary-color: default, $_box-shadow: false);
```

#### Need to tweak a shadow?

- Go to `settings.shadows.scss`
