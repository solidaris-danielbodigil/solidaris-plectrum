# Layout Object Documentation

The layout object `.o-layout`, provides classes to improve and create faster and consistent layouts.<br/>
This block has `3 modifiers`:

## Full Height
`.o-layout--full-height` <br/>
In order to use a vertical grid with `.o-flex--y`, it's ancestor needs to have a height set. <br/>
As fixing container's height is not recommended, `.o-layout--full-height` will set it to a relative value :`height: 100%`.<br/>

> **`.o-layout--full-height` needs to be added to all `.o-flex__item` ancestors until a container with a height is reached, or in the contrary up to the `html` tag.**

```html
<html class="o-layout o-layout--full-height">
  <body class="o-layout o-layout--full-height">
    <main class="o-layout o-layout--full-height">
      <section class="o-layout o-layout--full-height">
        <div class="o-flex o-flex--y o-layout o-layout--full-height">
          <div class="o-flex__item"></div>
        </div>
      </section>
    </main>
  </body>
</html>
```
