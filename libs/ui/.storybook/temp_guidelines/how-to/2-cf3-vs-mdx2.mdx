# CSF 3 vs MDX 2

Since `v7`, `.stories.mdx` have been deprecated in favor of `.stories.ts`. <br/>
It is a separation of concerns: `CSF` is used to render the component stories and their canvases, tests and other actions, while
`.mdx` files are now dedicated to document `.stories.ts`, as attached custom doc, or as standalone doc like this very file.

See more info here: <br/>
https://storybook.js.org/blog/storybook-7-docs/ <br/>
https://storybook.js.org/docs/react/migration-guide?ref=storybookblog.ghost.io#storiesmdx-to-mdxcsf
