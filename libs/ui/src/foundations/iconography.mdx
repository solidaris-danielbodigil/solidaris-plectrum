# Iconography

The Solidaris Design System uses **Bootstrap Icons** as its default icon set.
All icons are available via the `<sds-icon>` component:

```html
<!-- Bootstrap Icon (default) -->
<sds-icon icon="bi bi-house" />
<sds-icon icon="bi bi-house" size="lg" />

<!-- Non-decorative (standalone) -->
<sds-icon icon="bi bi-bell" label="Notifications" />

<!-- Custom SVG from registry -->
<sds-icon icon="logo-solidaris" source="svg" size="xl" />
```

Custom icons are exported from Figma as SVG and registered at app startup:

```ts
// app.config.ts
const registry = inject(IconRegistry);
registry.register('logo-solidaris', rawSvgString);
```
